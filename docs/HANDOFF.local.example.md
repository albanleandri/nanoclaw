# Local Handoff Example

This file is a template for `docs/HANDOFF.local.md`, which is intentionally ignored.

## Current objective
- Briefly state the active local objective.

## Private context
- Record any local-only assumptions, environment details, or private customization notes that should not be committed.

## Files changed
- List files touched in the current local work.

## Commands run
- List the important commands already run.

## Test/lint status
- Record what was verified and what was not.

## Open issues / next steps
- Note blockers, follow-ups, and anything the next agent should pick up.
